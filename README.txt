HOW TO ADD YOUR OWN LINK
1. Open index.html.
2. Find:
   href="YOUR-SAFE-LINK-HERE"
3. Replace only YOUR-SAFE-LINK-HERE with your legitimate website URL.
   Example:
   href="https://example.com"

HOW TO PUBLISH WITH VERCEL
1. Go to vercel.com and sign in.
2. Create a new project.
3. Upload this folder or connect a GitHub repository containing these files.
4. Deploy.
